# Paragon
A 1.12.2 Anarchy Client

Created by Wolfsurge and TeleTofu

# Build instructions
Windows:

`gradlew setupDecompWorkspace`

`gradlew build`

Linux/Mac:

`chmod +x gradlew`

`./gradlew setupDecompWorkspace`

`./gradlew build`
